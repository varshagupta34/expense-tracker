# Expense Tracker

A simple Angular expense tracker built with routing, reactive forms, services, BehaviorSubject, and currency formatting.

## Features

- Add expenses by category
- Delete expenses
- View total spent
- Filter expenses by category
- Reactive form validation
- Angular routing between pages
- Shared state using BehaviorSubject
- Currency display using Angular CurrencyPipe

## Tech Stack

- Angular
- TypeScript
- RxJS
- Angular Router
- Reactive Forms
- Angular Common

## Project Structure

- `main.ts` — app bootstrap and routes
- `ExpenseService` — stores and updates expense data
- `ExpensesPage` — shows list, filter, delete, and total
- `AddExpensePage` — reactive form for adding expenses

## How to Run

1. Open the project in StackBlitz or clone it locally.
2. Install dependencies if needed.
3. Run the Angular app.
4. Use the navigation links to switch between pages.

## Usage

- Go to **Expenses** to view all saved expenses.
- Use the category filter to narrow the list.
- Click **Delete** to remove an expense.
- Go to **Add Expense** to submit a new item.

## Learning Goals

This project demonstrates:
- Angular standalone components
- Angular Router
- Reactive Forms
- Services
- BehaviorSubject
- Pipes
- Structural directives like `@for` and `@empty`
